# Security Policy

## Reporting a Vulnerability

We take the security of ADQA seriously. If you discover a security vulnerability, especially related to PII detection or data handling, please let us know.

Please **do not** open a public issue. Instead, email your report to mohamed.talaat.ismael@gmail.com.

### What to include in your report:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.

We will acknowledge your report within 48 hours and provide a timeline for a fix.

## Supported Versions

Only the latest version of ADQA is supported for security updates. We recommend always running the most recent release.
